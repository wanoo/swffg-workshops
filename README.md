# Crafting Workshops — Star Wars FFG

*[Version française plus bas](#-version-française)*

A set of interactive crafting workshops for the
[Star Wars FFG](https://foundryvtt.com/packages/starwarsffg) system on Foundry VTT (v12/v13).
Fully bilingual **English / French** — content installs in your world's language.

## ⚒️ Crafting workshops
- **Lightsaber forge / recustom** — compendium-driven hilts, 24 crystals, 17 attachments;
  the Mechanics check comes FIRST and its result decides the outcome.
- **Potions & Talismans** (Force alchemy) — 17 recipes, contextual advantage/triumph
  spending, item created on the character.
- **Tuning / attachments** — install attachments (difficulty by rarity) and **activate
  mods with the book-accurate difficulty escalation** (Average +1 per mod already active
  on that attachment), tracked natively on the item.

## Install
1. Install the module (manifest URL or zip) and enable it in your world.
2. On first load as GM, accept the install prompt — it creates the crafting folder and
   crafting macros **in your world's
   language** (English or French).
3. Re-install anytime from *Configure Settings → Crafting Workshops*.

# 🇫🇷 Version française

Des ateliers d'artisanat interactifs pour le système
[Star Wars FFG](https://foundryvtt.com/packages/starwarsffg) sur Foundry VTT (v12/v13).
Entièrement bilingue **français / anglais** — le contenu s'installe dans la langue du monde.

## ⚒️ Ateliers d'artisanat
- **Forge / recustom de sabre laser** — poignées du compendium, 24 cristaux, 17 accessoires ;
  le jet de Mécanique vient EN PREMIER et son résultat décide de l'issue.
- **Potions & Talismans** (alchimie de la Force) — 17 recettes, dépense contextuelle des
  avantages/triomphes, objet créé sur la fiche.
- **Réglage / accessoires** — pose d'accessoires (difficulté selon rareté) et **activation
  de mods avec l'escalade de difficulté du livre** (Moyenne +1 par mod déjà activé sur
  l'accessoire), suivie nativement sur l'objet.

## Installation
1. Installe le module (URL de manifest ou zip) puis active-le dans ton monde.
2. Au premier chargement en MJ, accepte l'installation : macros d'artisanat sont créées **dans la
   langue du monde** (français ou anglais).
3. Réinstallable à tout moment via *Configuration des options → Crafting Workshops*.
